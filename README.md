# 💼 Portfolio Website – Kadir Kahraman

Eine moderne und persönliche Portfolio-Webseite, die meine Kenntnisse, Projekte, meinen Werdegang und meine technischen Fähigkeiten als Softwareentwickler präsentiert.

Das Projekt dient als digitale Visitenkarte und zentrale Anlaufstelle für Recruiter, Unternehmen und Interessierte, die mehr über mich und meine Arbeit erfahren möchten.

---

## 🚀 Features

- 👨‍💻 Persönliche Vorstellung (About Me)
- 📸 Profilbereich mit Bild
- 🛠️ Übersicht des Tech Stacks
- 📄 Download des Lebenslaufs als PDF
- 🔗 Verlinkung zu GitHub und LinkedIn
- 📊 Informationen zu Ausbildung, Interessen und beruflichem Werdegang
- 🎯 Aktuelle berufliche Ziele
- 💡 Persönliche Fun Facts
- 📱 Responsives Layout
- 🎨 Modernes Dark-Mode Design
- ⚡ Dynamische Inhalte mit JavaScript Templates
